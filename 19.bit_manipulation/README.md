# 位运算

## 异或（XOR）

异或：相同为 0，不同为 1.也可用“不进位加法”来理解。

异或操作的一些特点：

```
x ^ 0 = x
x ^ 1s = ~x // 注意 1s = ~0
x ^ (~x) = 1s
x ^ x = 0
c = a ^ b => a ^ c = b, b ^ c = a   // 交换两个数
a ^ b ^ c = a ^ (b ^ c) = (a ^ b) ^ c   // associative
```

## 指定位置的位运算

1. 将 x 最右边的 n 位清零：`x&(~0<<n)`
2. 获取 x 的第 n 位值（0或者1）：`(x>>n)&1`
3. 获取 x 的第 n 位的幂值
